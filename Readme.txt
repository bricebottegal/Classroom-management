Date : 2011
Contexte : création d'une solution de gestion des salles pour l'université du Havre
Licence : GNU GPL
Description : gestion Planning est une application permettant de gérer facilement le planning des réservations dans une ou plusieurs salles. Il existe 2 groupes distincts, les professeurs et les administrateurs. Les professeurs une fois identifiés peuvent demander une réservation. La période réservée passe alors en attente, un email est envoyé ou non selon la configuration du script à l'administrateur l'informant de la demande. Celui-ci valide ou refuse la demande de réservation et le professeur est informé de sa décision avec un email. Une partie administration protégée permet d'ajouter, modifier et supprimer les utilisateurs ainsi que les différentes salles.
